Today we are going to learn about the Number System

The number system or the numeral system is the system of naming or representing numbers. There are various types of number systems in maths like binary, decimal, etc.

What is the Number System in Maths?

A number system is defined as a system of writing for expressing numbers. It is the mathematical notation for representing numbers of a given set by using digits or other symbols in a consistent manner. It provides a unique representation of every number and represents the arithmetic and algebraic structure of the figures. It also allows us to operate arithmetic operations like addition, subtraction, multiplication and division.
